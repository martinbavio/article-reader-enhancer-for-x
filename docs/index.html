Privacy Policy for Article Reader Enhancer for X

Last Updated: January 13, 2026

Summary: This Extension does not collect any user data. All functionality operates locally in your browser.

Overview

Article Reader Enhancer for X ("the Extension") is committed to protecting your privacy. This privacy policy explains our data practices.

Data Collection

We do not collect, store, or transmit any user data.

The Extension:

  • Does NOT collect personal information
  • Does NOT track browsing history
  • Does NOT store user credentials
  • Does NOT transmit data to external servers
  • Does NOT use cookies or analytics
  • Does NOT sell or share user data with third parties

What the Extension Does

The Extension operates entirely locally in your browser to:

  1. Enhance Typography - Applies custom fonts and styling to article content on X (formerly Twitter)
  2. Improve Layout - Adjusts spacing, margins, and content width for better readability
  3. Hide Distractions - Removes sidebar, replies, and other UI elements using CSS
  4. Display Progress - Shows a reading progress indicator at the top of the page

All processing happens locally in your browser. No information ever leaves your device.

Third-Party Services

The Extension loads web fonts from Google Fonts CDN (fonts.googleapis.com) to display the Libre Baskerville typeface. This is a standard web font loading practice. Google's privacy policy applies to font requests: https://policies.google.com/privacy

The Extension does not use Google Fonts API for tracking or analytics - it only loads font files for display purposes.

Permissions Explanation

Host Permissions (twitter.com, x.com)

Required to inject CSS and JavaScript that enhance the visual presentation of article content on X. These permissions allow the Extension to:

  • Apply typography improvements
  • Modify page layout
  • Hide distracting UI elements
  • Display the reading progress indicator

All modifications are visual only and performed locally in your browser.

Data Security

Since the Extension does not collect, store, or transmit any data, there is no user data that could be compromised.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date.

Contact

If you have questions about this privacy policy, please contact us via the Chrome Web Store listing or file an issue on our GitHub repository.

Consent

By using the Extension, you consent to this privacy policy.
